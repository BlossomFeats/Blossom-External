# How to run

Windows
```py
cd your_directory

pip install -r requirements.txt

Run the NGNC.exe
```

Linux
```py
cd your_directory

pip3 install -r requirements.txt

chmod +x NGNC

./NGNC
```

**WORKS ONLY WITH PYTHON 3.5 AND ABOVE**
